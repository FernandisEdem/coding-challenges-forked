Coding Challenge: Style Your Web Page (Inline Styling)

Challenge:
Apply inline CSS to your HTML web page. Change the font, colors, and add a background color.

Instructions:
1. Open your code editor and create a new folder for this challenge. Name it "style-web-page-challenge-inline."

2. Inside the "style-web-page-challenge-inline" folder, create an HTML file named "index.html."

3. In "index.html," set up the basic HTML structure, including the following:
   - Document type declaration: <!DOCTYPE html>
   - Opening and closing <html> tags.
   - Opening and closing <head> tags.
   - A <title> element with the text "Style Your Web Page Challenge."
   - Opening and closing <body> tags.

4. Inside the <body> section, apply inline styles to the elements to achieve the following:
   - Change the font for headings and paragraphs.
   - Customize the colors for text and backgrounds.
   - Add a background color to the entire page.

5. Use the "style" attribute within HTML tags to apply inline styles.

6. Save your "index.html" file.

7. Now, open your "index.html" file in a web browser to see your styled web page with inline CSS.

8. Once you're finished, compare your code to the provided "final" solution in the challenge description to ensure everything matches.

Feel free to customize the content and wording of the instructions as needed. This .txt file provides clear steps for students to follow when completing the challenge using inline styles.
